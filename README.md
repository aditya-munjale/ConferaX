# Real-Time Video Conferencing & Chat Application

A real-time video conferencing and chat application built using the **MERN stack**, **WebRTC**, and **Socket.IO**. The platform allows users to create or join meetings using a unique meeting ID and communicate through video, audio, and live chat with low latency.

## Features
- Create or join meetings using a unique meeting ID  
- Real-time video and audio communication (WebRTC â€“ P2P)  
- Live chat during meetings  
- Instant user join/leave updates using Socket.IO  
- Guest user support  

## Tech Stack
- **Frontend:** React.js, JavaScript, HTML, CSS  
- **Backend:** Node.js, Express.js  
- **Real-Time Communication:** WebRTC, Socket.IO  
- **Database:** MongoDB  

## Overview
This project showcases practical experience in building real-time applications, peer-to-peer communication, and full-stack web development.
